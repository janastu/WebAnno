Getting Started

To use Annotorious, link the annotorious.css stylesheet into the <head> of your HTML page.

<link rel="stylesheet" type="text/css" href="annotorious.css" />

Include the annotorious.min.js script at the end of the document <body>.

<script type="text/javascript" src="annotorious.min.js"></script>
